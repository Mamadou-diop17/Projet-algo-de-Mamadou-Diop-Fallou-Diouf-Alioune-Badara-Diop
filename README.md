# Projet Algorithmique et Programmation 3
## Gestion des Supports de Cours

### 1. Présentation
Ce projet a été réalisé dans le cadre du module
Algorithmique et Programmation 3.
Il permet de gérer des supports de cours à l’aide
des structures de données vues en cours.

### 2. Objectifs
- Manipuler les structures de données
- Appliquer les listes chaînées
- Utiliser une pile (stack)
- Utiliser une file (queue)
- Concevoir un menu interactif en C

### 3. Structures de données utilisées
- Structure Support : représente un support de cours
- Liste chaînée : stockage des supports
- Pile : historique des actions effectuées
- File : notifications des opérations

### 4. Fonctionnalités
- Ajouter un support de cours
- Afficher la liste des supports
- Afficher l’historique des actions (pile)
- Afficher les notifications (file)
- Quitter le programme

### 5. Outils utilisés
- Langage : C
- Compilateur : GCC
- Système : Linux / Windows

### 6. Compilation et exécution
Compilation :
gcc app.c -o app

Exécution :
./app

### 7. Conclusion
Ce projet nous a permis de mieux comprendre
l’utilisation des structures de données en langage C
et leur importance dans la conception d’algorithmes.
